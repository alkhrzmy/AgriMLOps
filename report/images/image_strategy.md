## How to Create Images

### Option 1: Manual Drawing Tools
- Use draw.io (https://app.diagrams.net/) for architecture diagrams
- Use Lucidchart for flowcharts
- Use PowerPoint/Keynote for simple diagrams

### Option 2: Code-Based Diagrams
- Use Mermaid.js for flowcharts and sequence diagrams
- Use Python matplotlib/seaborn for charts/graphs
- Use PlantUML for architecture diagrams
