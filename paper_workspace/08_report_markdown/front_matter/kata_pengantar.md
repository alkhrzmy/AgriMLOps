# KATA PENGANTAR

Puji syukur penulis panjatkan ke hadirat Allah SWT atas segala rahmat dan karunia-Nya sehingga penulis dapat menyelesaikan karya tulis ilmiah ini yang berjudul "IMPLEMENTASI SISTEM DIAGNOSA PENYAKIT TANAMAN DENGAN MACHINE LEARNING OPERATIONS (MLOPS) DAN ACTIVE LEARNING" sebagai salah satu syarat untuk menyelesaikan tugas mata kuliah.

Penulisan karya tulis ilmiah ini bertujuan untuk mengimplementasikan sistem diagnosis penyakit tanaman berbasis deep learning dengan mengintegrasikan Machine Learning Operations (MLOps) lifecycle dan active learning untuk pengelolaan model yang berkelanjutan. Penyakit tanaman merupakan salah satu ancaman terbesar dalam pertanian yang dapat menyebabkan kerugian ekonomi yang signifikan. Metode tradisional diagnosis penyakit tanaman bergantung pada keahlian manual yang memiliki keterbatasan dalam hal skalabilitas dan aksesibilitas.

Perkembangan teknologi deep learning dan computer vision telah membuka peluang baru untuk diagnosis penyakit tanaman yang otomatis dan akurat. Namun, sebagian besar penelitian sebelumnya fokus pada pengembangan model tanpa mempertimbangkan aspek operasional seperti monitoring, retraining, dan pengelolaan model dalam produksi. Machine Learning Operations (MLOps) muncul sebagai disiplin yang menggabungkan praktik DevOps dengan machine learning untuk mengelola siklus hidup model secara end-to-end.

Penelitian ini mengimplementasikan sistem AgriMLOps yang lengkap, mencakup model inference dengan EfficientNetV2-B0, prediction logging, user feedback collection, active learning queue, monitoring dashboard, model registry, dan deployment pada cloud infrastructure. Sistem ini dilatih dalam tiga iterasi: v1 (baseline), v2 (retraining tanpa feedback), dan v3 (active learning retraining dengan 9 validated feedback samples). Model v3 mencapai accuracy 0.8288 dan macro F1 0.8127, dengan improvement Best Val F1 sebesar +0.0147 dibanding v2.

Penulis menyadari bahwa karya tulis ilmiah ini masih jauh dari sempurna dan masih banyak kekurangan. Oleh karena itu, penulis mengharapkan kritik dan saran yang membangun dari berbagai pihak demi perbaikan di masa mendatang. Semoga karya tulis ilmiah ini dapat bermanfaat bagi pembaca dan memberikan kontribusi bagi pengembangan teknologi pertanian berbasis kecerdasan buatan di Indonesia.
