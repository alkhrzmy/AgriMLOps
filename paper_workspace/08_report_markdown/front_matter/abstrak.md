# ABSTRAK

Penyakit tanaman merupakan salah satu ancaman terbesar dalam pertanian yang dapat menyebabkan kerugian hasil panen hingga 40% secara global setiap tahunnya. Metode tradisional diagnosis penyakit tanaman bergantung pada keahlian manual yang memiliki keterbatasan dalam skalabilitas dan aksesibilitas. Perkembangan teknologi deep learning telah membuka peluang baru untuk diagnosis penyakit tanaman yang otomatis dan akurat, namun sebagian besar penelitian sebelumnya fokus pada pengembangan model tanpa mempertimbangkan aspek operasional dalam produksi.

Penelitian ini bertujuan untuk mengimplementasikan sistem diagnosis penyakit tanaman berbasis deep learning dengan mengintegrasikan Machine Learning Operations (MLOps) lifecycle dan active learning untuk pengelolaan model yang berkelanjutan. Sistem yang dikembangkan menggunakan EfficientNetV2-B0 dengan dataset PlantWild subset (15 kelas, 5.645 gambar) untuk klasifikasi penyakit tanaman in-the-wild. MLOps lifecycle diimplementasikan melalui prediction logging, user feedback collection, active learning queue, monitoring dashboard, dan model registry. Active learning queue secara otomatis menambahkan prediksi dengan confidence rendah untuk validasi, kemudian data yang divalidasi digunakan untuk retraining model secara iteratif.

Sistem dilatih dalam tiga iterasi: v1 (baseline dengan ImageNet pretrained), v2 (retraining tanpa feedback), dan v3 (active learning retraining dengan 9 validated feedback samples sebagai transfer learning dari v2). Model v3 mencapai accuracy 0.8288 (+0.0071 vs v1, +0.0035 vs v2), macro F1 0.8127 (+0.0041 vs v1, +0.0003 vs v2), dan Best Val F1 0.8254 (+0.0147 vs v2). Sistem berhasil dideploy pada DigitalOcean Droplet dengan Docker Compose, menyediakan akses web (Streamlit) dan API (FastAPI) yang skalabel.

Hasil penelitian menunjukkan bahwa MLOps lifecycle dapat diimplementasikan secara efektif untuk pengelolaan model machine learning dalam konteks diagnosis penyakit tanaman. Active learning dengan validated feedback menunjukkan potensi improvement dalam generalisasi model, dengan peningkatan Best Val F1 yang signifikan. Sistem yang dikembangkan mengisi gap literatur dengan mengintegrasikan komponen MLOps (inference, logging, feedback, active learning, monitoring, model registry, deployment) ke dalam satu platform khusus diagnosis penyakit tanaman in-the-wild.
