## Sintesis Research Gap

Mayoritas penelitian berfokus pada pengembangan model atau benchmark dataset, sedangkan penelitian aplikasi web yang ada belum mengintegrasikan lifecycle operasional model secara lengkap. AgriMLOps memposisikan kontribusinya sebagai platform implementatif yang menggabungkan inference, logging, user feedback, active learning queue, model registry, monitoring, dan deployment cloud untuk diagnosis penyakit tanaman berbasis PlantWild subset. Klaim penelitian dibatasi pada active learning berbasis controlled validated feedback simulation, bukan feedback petani asli, bukan true open-world detection, bukan domain adaptation eksplisit, dan bukan multimodal retrieval.
